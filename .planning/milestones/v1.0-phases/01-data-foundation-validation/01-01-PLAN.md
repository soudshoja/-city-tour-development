  <action>
    Create two migrations using `php artisan make:migration`:

    **bulk_uploads table:**
    ```
    - id (primary key)
    - company_id (unsignedBigInteger, foreign key to companies)
    - agent_id (unsignedBigInteger, foreign key to agents)
    - user_id (unsignedBigInteger, foreign key to users)
    - original_filename (string) — original uploaded file name
    - stored_path (string) — path in storage/app where file is kept
    - status (enum: 'pending', 'validating', 'validated', 'processing', 'completed', 'failed')
    - total_rows (unsignedInteger, default 0)
    - valid_rows (unsignedInteger, default 0)
    - error_rows (unsignedInteger, default 0)
    - flagged_rows (unsignedInteger, default 0) — rows with unknown clients
    - error_summary (json, nullable) — summary of validation errors
    - timestamps
    - softDeletes
    ```

    **bulk_upload_rows table:**
    ```
    - id (primary key)
    - bulk_upload_id (unsignedBigInteger, foreign key to bulk_uploads, cascadeOnDelete)
    - row_number (unsignedInteger) — Excel row number (1-indexed, excluding header)
    - status (enum: 'valid', 'error', 'flagged')
    - task_id (unsignedBigInteger, nullable) — matched task ID
    - client_id (unsignedBigInteger, nullable) — matched client ID
    - supplier_id (unsignedBigInteger, nullable) — matched supplier ID
    - raw_data (json) — original row data from Excel
    - errors (json, nullable) — array of validation error messages for this row
    - flag_reason (string, nullable) — reason if flagged (e.g., 'unknown_client')
    - timestamps
    ```

    **BulkUpload model** (`app/Models/BulkUpload.php`):
    - Use `HasFactory, SoftDeletes` traits
    - Define `$fillable` with all columns
    - Define `$casts`: `error_summary => 'array'`, `status => 'string'`
    - Relationships: `belongsTo(Company)`, `belongsTo(Agent)`, `belongsTo(User)`, `hasMany(BulkUploadRow)`
    - Scope: `scopeForCompany($query, $companyId)` — filters by company_id

    **BulkUploadRow model** (`app/Models/BulkUploadRow.php`):
    - Use `HasFactory` trait (no soft deletes — rows cascade with parent)
    - Define `$fillable` with all columns
    - Define `$casts`: `raw_data => 'array'`, `errors => 'array'`
    - Relationships: `belongsTo(BulkUpload)`, `belongsTo(Task)`, `belongsTo(Client)`, `belongsTo(Supplier)`

    Run `php artisan migrate` to verify migrations execute without errors.
