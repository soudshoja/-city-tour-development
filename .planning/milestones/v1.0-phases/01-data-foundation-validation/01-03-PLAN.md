    **BulkInvoiceController::upload()** — Add an `upload` method to the existing controller (created in 01-01):

    Method signature: `public function upload(BulkInvoiceUploadRequest $request): JsonResponse`

    Orchestration flow:
    1. **Get context**: `$user = Auth::user()`, `$companyId = getCompanyId($user)`, resolve `agent_id` from user's agent relationship
    2. **Store file**: `$file = $request->file('file')`, store to `bulk-uploads/{$companyId}/{timestamp}_{original_name}` using `Storage::disk('local')->putFileAs('bulk-uploads/' . $companyId, $file, time() . '_' . $file->getClientOriginalName())`
    3. **Parse Excel**: `$rows = Excel::toArray(new BulkInvoiceImport, $file)[0]` — gets first sheet as array of associative arrays
    4. **Validate headers**: `$headerResult = $validationService->validateHeaders(array_keys($rows[0] ?? []))` — if `$headerResult['valid'] === false`, create BulkUpload with status 'failed' and return 422 with missing headers error
    5. **Validate all rows**: `$results = $validationService->validateAll($rows, $companyId)`
    6. **Create BulkUpload record**:
       ```php
       $bulkUpload = BulkUpload::create([
           'company_id' => $companyId,
           'agent_id' => $agentId,
           'user_id' => $user->id,
           'original_filename' => $file->getClientOriginalName(),
           'stored_path' => $storedPath,
           'status' => $results['errors'] > 0 ? 'validated' : 'validated',
           'total_rows' => $results['total'],
           'valid_rows' => $results['valid'],
           'error_rows' => $results['errors'],
           'flagged_rows' => $results['flagged'],
           'error_summary' => $this->buildErrorSummary($results['rows']),
       ]);
       ```
    7. **Create BulkUploadRow records** — Use bulk insert for performance:
       ```php
       $rowRecords = [];
       foreach ($results['rows'] as $index => $rowResult) {
           $rowRecords[] = [
               'bulk_upload_id' => $bulkUpload->id,
               'row_number' => $index + 1,
               'status' => $rowResult['status'],
               'task_id' => $rowResult['matched']['task_id'] ?? null,
               'client_id' => $rowResult['matched']['client_id'] ?? null,
               'supplier_id' => $rowResult['matched']['supplier_id'] ?? null,
               'raw_data' => json_encode($rows[$index]),
               'errors' => !empty($rowResult['errors']) ? json_encode($rowResult['errors']) : null,
               'flag_reason' => $rowResult['flag_reason'] ?? null,
               'created_at' => now(),
               'updated_at' => now(),
           ];
       }
       BulkUploadRow::insert($rowRecords);
       ```
    8. **Return JSON response**:
       ```php
       return response()->json([
           'upload_id' => $bulkUpload->id,
           'status' => $bulkUpload->status,
           'summary' => [
               'total_rows' => $results['total'],
               'valid_rows' => $results['valid'],
               'error_rows' => $results['errors'],
               'flagged_rows' => $results['flagged'],
           ],
           'has_errors' => $results['errors'] > 0,
           'has_flagged' => $results['flagged'] > 0,
       ]);
       ```

    **Private helper method** `buildErrorSummary(array $rowResults): array`:
    - Aggregate errors by type (e.g., "missing_field: 5 rows", "unknown_supplier: 3 rows")
    - Return array of `['type' => 'count']` pairs for the error_summary JSON field

    **Constructor injection**: Add `BulkUploadValidationService $validationService` to the constructor and store as property.
