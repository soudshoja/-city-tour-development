  <action>
Add a `preview(int $id)` method to BulkInvoiceController that:

1. Gets authenticated user and companyId via `getCompanyId($user)`
2. Loads BulkUpload scoped by company_id AND status='validated' with eager loading:
   `BulkUpload::where('id', $id)->where('company_id', $companyId)->where('status', 'validated')->with(['rows.client', 'rows.supplier'])->firstOrFail()`
3. Separates valid rows from flagged rows:
   - `$validRows = $bulkUpload->rows->where('status', 'valid')`
   - `$flaggedRows = $bulkUpload->rows->where('status', 'flagged')`
4. Groups valid rows by composite key (client_id + invoice_date):
   ```php
   $invoiceGroups = $validRows->groupBy(function($row) {
       $clientId = $row->client_id;
       $invoiceDate = $row->raw_data['invoice_date'] ?? date('Y-m-d');
       return "{$clientId}_{$invoiceDate}";
   });
   ```
5. Counts unique clients: `$clientCount = $validRows->pluck('client_id')->unique()->count()`
6. Returns view: `return view('bulk-invoice.preview', compact('bulkUpload', 'invoiceGroups', 'flaggedRows', 'clientCount'))`

Add `use Illuminate\View\View;` import at top of controller.

In routes/web.php, add inside the existing `bulk-invoices` route group (after the error-report route):
```php
Route::get('/{id}/preview', [BulkInvoiceController::class, 'preview'])->name('preview');
```

IMPORTANT: Also update the existing `upload()` method to return a redirect to the preview page on successful validation instead of JSON. Change the success response at the end of upload() from `return response()->json(...)` to:
```php
return redirect()->route('bulk-invoices.preview', $bulkUpload->id)
    ->with('message', "Upload validated: {$results['valid']} valid rows, {$results['errors']} errors, {$results['flagged']} flagged.");
```

Also update the upload() method signature to return `JsonResponse|RedirectResponse` (add RedirectResponse import if not already present).
