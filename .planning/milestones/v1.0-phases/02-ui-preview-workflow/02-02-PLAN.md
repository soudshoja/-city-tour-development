Add three methods to BulkInvoiceController:

**approve(int $id): RedirectResponse**
1. Get user and companyId via `getCompanyId($user)`
2. Use conditional update to prevent race conditions (double-click, concurrent requests):
   ```php
   $updated = BulkUpload::where('id', $id)
       ->where('company_id', $companyId)
       ->where('status', 'validated')
       ->update(['status' => 'processing']);
   ```
3. If `$updated === 0`, redirect back with error: `return redirect()->back()->withErrors(['error' => 'Upload already processed or no longer in validated status.']);`
4. Load the BulkUpload for summary data: `$bulkUpload = BulkUpload::findOrFail($id);`
5. Redirect to success page: `return redirect()->route('bulk-invoices.success', $id)->with('message', 'Invoices approved for creation.');`

**reject(int $id): RedirectResponse**
1. Get user and companyId
2. Use conditional update:
   ```php
   $updated = BulkUpload::where('id', $id)
       ->where('company_id', $companyId)
       ->where('status', 'validated')
       ->update(['status' => 'rejected']);
   ```
3. If `$updated === 0`, redirect back with error: `return redirect()->back()->withErrors(['error' => 'Upload already processed or no longer in validated status.']);`
4. Redirect to upload page (use a sensible route -- for now redirect to dashboard with flash): `return redirect()->route('dashboard')->with('message', 'Upload rejected and discarded.');`

**success(int $id): View**
1. Get user and companyId
2. Load BulkUpload scoped by company_id: `BulkUpload::where('id', $id)->where('company_id', $companyId)->firstOrFail()`
3. Count invoice groups for display (same grouping logic as preview, but just the count):
   - Load valid rows: `$validRows = $bulkUpload->rows()->where('status', 'valid')->with('client')->get()`
   - Group by composite key: same groupBy as preview()
   - `$invoiceCount = $invoiceGroups->count()`
   - `$clientCount = $validRows->pluck('client_id')->unique()->count()`
4. For now, invoices collection is empty (Phase 3 creates actual invoices): `$invoices = collect([])`
5. Return view: `return view('bulk-invoice.success', compact('bulkUpload', 'invoiceCount', 'clientCount', 'invoices'))`

Add `use Illuminate\View\View;` import if not already present.

In routes/web.php, add inside the existing `bulk-invoices` route group:

```php
Route::post('/{id}/reject', [BulkInvoiceController::class, 'reject'])->name('reject');
Route::get('/{id}/success', [BulkInvoiceController::class, 'success'])->name('success');
```
