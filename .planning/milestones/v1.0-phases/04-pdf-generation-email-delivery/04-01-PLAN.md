  <action>
    Create a new Mailable class `BulkInvoicesMail` at `app/Mail/BulkInvoicesMail.php`.

    **Constructor:** Accept `int $bulkUploadId` (not Eloquent model â€” serialize only ID per anti-pattern from research).

    **build() method:**
    1. Load BulkUpload with eager loading: `BulkUpload::with('agent.branch.company')->findOrFail($this->bulkUploadId)`
    2. Load invoices: `Invoice::whereIn('id', $bulkUpload->invoice_ids)->with('client', 'agent.branch.company', 'invoiceDetails.task.supplier')->get()`
    3. Set subject: `"Bulk Invoice Upload - {$invoiceCount} Invoices Created"`
    4. Use view: `email.bulk-invoices`
    5. Pass view data: `invoices`, `bulkUpload`, `company` (from `$bulkUpload->agent->branch->company`)

    **attachments() method (Laravel 11 pattern):**
    For each invoice in the loaded collection:
    1. Generate PDF in memory using `Pdf::loadView('invoice.pdf.invoice', [...])` with `setPaper('a4', 'portrait')`
    2. Pass view data matching the existing InvoiceMail pattern: `invoice`, `company`, `invoiceDetails`, `isPdf => true`
    3. Return `Attachment::fromData(fn () => $pdf->output(), "Invoice-{$invoice->invoice_number}.pdf")->withMime('application/pdf')`

    Use `Barryvdh\DomPDF\Facade\Pdf` (capital P facade, lowercase 'p' class).
    Use `Illuminate\Mail\Attachment` for the fromData pattern.
    Do NOT implement ShouldQueue on the Mailable itself â€” the SendInvoiceEmailsJob handles queueing.

    **Email Blade template** at `resources/views/email/bulk-invoices.blade.php`:
    Create a simple, professional HTML email (inline CSS, no external resources â€” DomPDF constraint).
    Content:
    - Header: "Invoice Delivery" with company name
    - Body: "X invoice(s) have been created from bulk upload: {filename}"
    - Invoice list table: Invoice Number | Client | Amount | Date
    - Footer: "PDFs are attached to this email." + "This is an automated message."

    Follow the pattern from `resources/views/email/notification.blade.php` for styling conventions.
  </action>

  <name>Task 2: Create SendInvoiceEmailsJob Queue Job</name>
  <files>

  <action>
    Create a new queue job `SendInvoiceEmailsJob` at `app/Jobs/SendInvoiceEmailsJob.php`.

    **Class structure:** Follow the existing `CreateBulkInvoicesJob` pattern exactly:
    - `implements ShouldQueue`
    - `use Dispatchable, InteractsWithQueue, Queueable, SerializesModels`
    - `public $tries = 3;`
    - `public $backoff = [10, 30, 60];`
    - `public $timeout = 300;` (PDF generation for many invoices can be slow)

    **Constructor:** `public function __construct(public int $bulkUploadId) {}`

    **handle() method:**
    1. Load BulkUpload: `BulkUpload::with('agent.branch.company')->findOrFail($this->bulkUploadId)`
    2. Verify status is 'completed' (guard clause â€” if not completed, log warning and return early)
    3. Verify invoice_ids is not empty (guard clause)
    4. Get company: `$company = $bulkUpload->agent->branch->company`
    5. Get agent: `$agent = $bulkUpload->agent`
    6. Set log context with bulk_upload_id and company_id

    **Send to accountant (company email):**
    ```php
    if ($company && $company->email) {
        Mail::to($company->email)
            ->queue(new BulkInvoicesMail($this->bulkUploadId));
        Log::info('Queued bulk invoice email to accountant', ['email' => $company->email]);
    } else {
        Log::warning('No company email found, skipping accountant notification', [
            'company_id' => $bulkUpload->company_id,
        ]);
    }
    ```

    **Send to uploading agent:**
    ```php
    if ($agent && $agent->email) {
        Mail::to($agent->email)
            ->queue(new BulkInvoicesMail($this->bulkUploadId));
        Log::info('Queued bulk invoice email to agent', ['email' => $agent->email]);
    } else {
        Log::warning('No agent email found, skipping agent notification', [
            'agent_id' => $bulkUpload->agent_id,
        ]);
    }
    ```

    **failed() method:**
    Log error with exception details. Do NOT update BulkUpload status â€” the upload itself succeeded, only email delivery failed. This is a non-critical failure.
    ```php
    Log::error('SendInvoiceEmailsJob failed', [
        'bulk_upload_id' => $this->bulkUploadId,
        'exception' => $exception->getMessage(),
    ]);
    ```

    Use `Illuminate\Support\Facades\Mail` and `Illuminate\Support\Facades\Log`.
    Import `App\Mail\BulkInvoicesMail`.
