<task type="auto">
  <name>Task 2: Create DotwCacheService class</name>
  <files>app/Services/DotwCacheService.php</files>
  <action>
    Create app/Services/DotwCacheService.php with namespace App\Services. Follow PSR-12 and include full PHPDoc on every method.

    The class must implement these three public methods:

    **buildKey(int $companyId, string $destination, string $checkin, string $checkout, array $rooms): string**
    - Format: `{prefix}_{companyId}_{destination}_{checkin}_{checkout}_{roomsHash}`
    - prefix = config('dotw.cache.prefix') — defaults to 'dotw_search'
    - destination = strtolower(trim($destination)) — normalize city codes
    - checkin/checkout = passed as-is (already YYYY-MM-DD)
    - roomsHash = md5(json_encode($this->normalizeRooms($rooms))) where normalizeRooms sorts each room's keys, sorts children ages ascending, then sorts the rooms array itself by adultsCode ascending to produce a stable JSON regardless of array order

    **remember(string $key, callable $callback): array**
    - Calls Cache::remember($key, $ttl, $callback) where $ttl is new \DateInterval('PT'.$this->ttl.'S')
    - Returns the result of $callback if cache miss, or the cached value if cache hit
    - Does NOT add 'cached' flag here — callers must check isCached() to know if a hit occurred

    **isCached(string $key): bool**
    - Returns Cache::has($key)

    **forget(string $key): bool**
    - Returns Cache::forget($key)

    **Private helper normalizeRooms(array $rooms): array**
    - For each room: ksort($room), sort children ages array ascending, return normalized room
    - Sort the rooms array by adultsCode ascending to make multi-room configs order-independent

    Constructor:
    ```php
    public function __construct()
    {
        $this->ttl = (int) config('dotw.cache.ttl', 150);
        $this->prefix = config('dotw.cache.prefix', 'dotw_search');
    }
    ```

    Private properties: $ttl (int), $prefix (string).
