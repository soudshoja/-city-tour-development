  <name>Task 2: Create DotwTraceMiddleware and register it in Lighthouse</name>

  <action>
    Create app/GraphQL/Middleware/DotwTraceMiddleware.php. This is a standard Laravel HTTP middleware (implements handle($request, $next)) registered in Lighthouse's route middleware — NOT a Lighthouse field middleware.

    The middleware must:
    1. Generate a UUID v4 trace_id at the start of every request using Str::uuid()->toString().
    2. Store it on the request: $request->attributes->set('dotw_trace_id', $traceId).
    3. Bind it in the Laravel service container so resolvers can read it: app()->instance('dotw.trace_id', $traceId).
    4. Record the request start time in microseconds: $startTime = microtime(true).
    5. Call $next($request) to proceed.
    6. On the response, add headers:
       - X-Trace-ID: {trace_id}
       - X-Request-Time-Ms: {elapsed milliseconds, rounded to 2 decimal places}
    7. Return the modified response.

    Full class structure:
    ```php
    <?php

    declare(strict_types=1);

    namespace App\GraphQL\Middleware;

    use Closure;
    use Illuminate\Http\Request;
    use Illuminate\Support\Str;
    use Symfony\Component\HttpFoundation\Response;

    /**
     * DOTW Trace Middleware
     *
     * Injects a per-request UUID trace ID and timing headers into every
     * DOTW GraphQL response. Enables log correlation between N8N workflows,
     * Resayil WhatsApp conversations, and server-side audit logs.
     *
     * Sets:
     *  - X-Trace-ID: UUID v4 unique to this request
     *  - X-Request-Time-Ms: Total processing time in milliseconds
     *
     * The trace_id is also bound in the service container as 'dotw.trace_id'
     * so GraphQL resolvers can include it in their DotwMeta response object.
     *
     * @package App\GraphQL\Middleware
     */
    class DotwTraceMiddleware
    {
        /**
         * Handle an incoming request.
         *
         * @param Request $request
         * @param Closure(Request): Response $next
         * @return Response
         */
        public function handle(Request $request, Closure $next): Response
        {
            $traceId = Str::uuid()->toString();
            $startTime = microtime(true);

            // Make trace_id available to resolvers
            $request->attributes->set('dotw_trace_id', $traceId);
            app()->instance('dotw.trace_id', $traceId);

            /** @var Response $response */
            $response = $next($request);

            $elapsedMs = round((microtime(true) - $startTime) * 1000, 2);

            $response->headers->set('X-Trace-ID', $traceId);
            $response->headers->set('X-Request-Time-Ms', (string) $elapsedMs);

            return $response;
        }
    }
    ```

    Register in config/lighthouse.php: Add DotwTraceMiddleware to the 'middleware' array inside 'route'. Insert it BEFORE AcceptJson — trace capture must happen as early as possible:

