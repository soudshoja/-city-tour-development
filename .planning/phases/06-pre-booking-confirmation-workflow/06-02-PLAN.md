**Class structure:**

```php
<?php

declare(strict_types=1);

namespace App\GraphQL\Mutations;

use App\Models\DotwBooking;
use App\Models\DotwPrebook;
use App\Services\DotwAuditService;
use App\Services\DotwService;
use Illuminate\Support\Facades\DB;
use Illuminate\Support\Facades\Log;
use Illuminate\Support\Str;

class DotwCreatePreBooking
{
    public function __construct(
        private readonly DotwAuditService $auditService,
    ) {}

    public function __invoke(mixed $root, array $args): array
    {
        $input          = $args;
        $prebookKey     = $input['prebook_key'];
        $checkin        = $input['checkin'];
        $checkout       = $input['checkout'];
        $passengers     = $input['passengers'];
        $rooms          = $input['rooms'];
        $destination    = $input['destination'] ?? null;

        // Extract Resayil headers from GraphQL context
        $resayilMessageId = request()->header('X-Resayil-Message-ID');
        $resayilQuoteId   = request()->header('X-Resayil-Quote-ID');

        // --- Step 1: Load and validate prebook ---
        $prebook = DotwPrebook::where('prebook_key', $prebookKey)->first();

        if ($prebook === null) {
            return $this->errorResponse('VALIDATION_ERROR', 'Prebook not found.', 'RESEARCH');
        }

        $companyId = $prebook->company_id;

        if (! $prebook->isValid()) {
            return $this->errorResponse(
                'ALLOCATION_EXPIRED',
                'Rate offer expired, please search again.',
                'RESEARCH'
            );
        }

        // --- Step 2: Validate passengers ---
        // BOOK-02: passenger count matches room occupancy (total adults across all rooms)
        $expectedPassengerCount = array_sum(array_column($rooms, 'adults'));
        $validationError = $this->validatePassengers($passengers, $expectedPassengerCount);
        if ($validationError !== null) {
            return $validationError;
        }

        // --- Step 3: Build confirmBooking params ---
        // BOOK-04: call DotwService::confirmBooking() with reconstructed params
        $dotwService        = new DotwService($companyId);
        $leadPassengerEmail = $passengers[0]['email'];
        $customerReference  = (string) Str::uuid();

        // Format passengers per room for DOTW XML (salutation, firstName, lastName required by buildPassengersXml)
        $formattedPassengers = array_map(function ($p) {
            return [
                'salutation' => $p['salutation'],
                'firstName'  => $p['firstName'],
                'lastName'   => $p['lastName'],
            ];
        }, $passengers);

        // Reconstruct room params from prebook stored data + input rooms
        $firstRoom       = $rooms[0];
        $childrenAges    = $firstRoom['childrenAges'] ?? [];
        $adultsCount     = $firstRoom['adults'] ?? 2;

        $confirmParams = [
            'fromDate'            => $checkin,
            'toDate'              => $checkout,
            'currency'            => $prebook->original_currency ?: 'USD',
            'productId'           => $prebook->hotel_code,
            'sendCommunicationTo' => $leadPassengerEmail,
            'customerReference'   => $customerReference,
            'rooms' => [
                [
                    'roomTypeCode'      => $prebook->room_type,
                    'selectedRateBasis' => $prebook->room_rate_basis,
                    'allocationDetails' => $prebook->allocation_details,  // raw — never encode (Pitfall 1)
                    'adultsCode'        => $adultsCount,
                    'actualAdults'      => $adultsCount,
                    'children'          => $childrenAges,
                    'actualChildren'    => $childrenAges,
                    'beddingPreference' => 0,
                    'passengers'        => $formattedPassengers,
                ],
            ],
        ];

        // --- Step 4: Call DOTW confirmBooking + create booking atomically ---
        try {
            $confirmation = $dotwService->confirmBooking(
                $confirmParams,
                $resayilMessageId,
                $resayilQuoteId,
                $companyId
            );
            // confirmBooking() returns: ['bookingCode', 'confirmationNumber', 'status', 'paymentGuaranteedBy']

            $confirmationCode   = $confirmation['bookingCode'] ?? '';
            $confirmationNumber = $confirmation['confirmationNumber'] ?? '';
            $bookingStatus      = $confirmation['status'] ?? 'confirmed';

            $bookingDetails = [
                'hotel_code'   => $prebook->hotel_code,
                'hotel_name'   => $prebook->hotel_name ?? $prebook->hotel_code,
                'checkin'      => $checkin,
                'checkout'     => $checkout,
                'room_type'    => $prebook->room_type,
                'total_fare'   => $prebook->total_fare,
                'currency'     => $prebook->original_currency ?: 'USD',
                'is_refundable' => $prebook->is_refundable,
            ];

            // BOOK-06: atomic — expire prebook + create booking record (same pattern as BLOCK-08)
            $booking = null;
            DB::transaction(function () use (
                $prebook, $prebookKey, $confirmationCode, $confirmationNumber,
                $customerReference, $bookingStatus, $passengers, $bookingDetails,
                $resayilMessageId, $resayilQuoteId, $companyId, &$booking
            ) {
                // Expire the prebook to prevent double-booking
                $prebook->update(['expired_at' => now()]);

                $booking = DotwBooking::create([
                    'prebook_key'         => $prebookKey,
                    'confirmation_code'   => $confirmationCode,
                    'confirmation_number' => $confirmationNumber,
                    'customer_reference'  => $customerReference,
                    'booking_status'      => $bookingStatus,
                    'passengers'          => $passengers,
                    'hotel_details'       => $bookingDetails,
                    'resayil_message_id'  => $resayilMessageId,
                    'resayil_quote_id'    => $resayilQuoteId,
                    'company_id'          => $companyId,
                ]);
            });

            // BOOK-07: supplementary audit log with confirmation_code and booking_status
            // (DotwService::confirmBooking() already calls DotwAuditService::log(OP_BOOK, ...) internally.
            //  This supplementary log captures prebook_key + confirmation_code association explicitly — same two-phase pattern as BLOCK-07.)
            try {
                $this->auditService->log(
                    DotwAuditService::OP_BOOK,
                    ['prebook_key' => $prebookKey, 'customer_reference' => $customerReference],
                    ['confirmation_code' => $confirmationCode, 'booking_status' => $bookingStatus],
                    $resayilMessageId,
                    $resayilQuoteId,
                    $companyId
                );
            } catch (\Throwable) {
                // Fail-silent — audit failure must NEVER break booking response (established pattern)
            }

            // BOOK-05: return success response
            $leadPassenger = $passengers[0];
            $leadGuestName = trim(($leadPassenger['firstName'] ?? '') . ' ' . ($leadPassenger['lastName'] ?? ''));

            return [
                'success' => true,
                'error'   => null,
                'meta'    => $this->buildMeta($companyId),
                'cached'  => false,
                'data'    => [
                    'booking_confirmation_code' => $confirmationCode,
                    'booking_status'            => $bookingStatus,
                    'itinerary_details'         => [
                        'hotel_code'          => $prebook->hotel_code,
                        'hotel_name'          => $prebook->hotel_name ?? $prebook->hotel_code,
                        'checkin'             => $checkin,
                        'checkout'            => $checkout,
                        'room_type'           => $prebook->room_type,
                        'rate_basis'          => $prebook->room_rate_basis ?? '',
                        'total_fare'          => (float) $prebook->total_fare,
                        'currency'            => $prebook->original_currency ?: 'USD',
                        'is_refundable'       => (bool) $prebook->is_refundable,
                        'lead_guest_name'     => $leadGuestName,
                        'customer_reference'  => $customerReference,
                        'confirmation_number' => $confirmationNumber,
                    ],
                    'alternatives' => [],
                ],
            ];

        } catch (\RuntimeException $e) {
            // BOOK-08 / ERROR-03 / ERROR-04: DOTW returned rate unavailable, hotel sold out, or other specific error
            $message = $e->getMessage();
            Log::channel('dotw')->error('createPreBooking failed', [
                'prebook_key' => $prebookKey,
                'error'       => $message,
                'company_id'  => $companyId,
            ]);

            // ERROR-04 + ERROR-06: rate unavailable or hotel sold out — suggest 3 alternatives
            $isRateUnavailable = str_contains(strtolower($message), 'unavailable')
                || str_contains(strtolower($message), 'sold out')
                || str_contains(strtolower($message), 'no longer available');

            $alternatives = [];
            if ($isRateUnavailable && $destination !== null) {
                try {
                    $altResults = $dotwService->searchHotels([
                        'fromDate' => $checkin,
                        'toDate'   => $checkout,
                        'currency' => $prebook->original_currency ?: 'USD',
                        'filters'  => ['city' => $destination],
                        'rooms'    => [['adults' => $rooms[0]['adults'] ?? 2, 'children' => []]],
                    ]);
                    $alternatives = array_slice($altResults['hotels'] ?? [], 0, 3);
                } catch (\Throwable) {
                    $alternatives = [];  // Fail-silent — alternative search must not cascade into new error
                }
            }

            $errorCode    = $isRateUnavailable ? 'RATE_UNAVAILABLE' : 'API_ERROR';
            $errorMessage = $isRateUnavailable
                ? 'This hotel/rate no longer available. Please search again or choose an alternative.'
                : 'Booking failed. Please try again or contact support.';

            return [
                'success' => false,
                'error'   => $this->buildError($errorCode, $errorMessage, 'RESEARCH', $message),
                'meta'    => $this->buildMeta($companyId),
                'cached'  => false,
                'data'    => [
                    'booking_confirmation_code' => '',
                    'booking_status'            => 'failed',
                    'itinerary_details'         => $this->emptyItinerary($prebook, $checkin, $checkout),
                    'alternatives'              => $alternatives,
                ],
            ];

        } catch (\Exception $e) {
            // Generic error — credentials not configured or unexpected exception
            Log::channel('dotw')->error('createPreBooking unexpected error', [
                'prebook_key' => $prebookKey,
                'error'       => $e->getMessage(),
            ]);

            return $this->errorResponse(
                'API_ERROR',
                'Booking failed due to a system error. Please try again.',
                'RETRY'
            );
        }
    }

    /**
     * Validate passenger count and field completeness.
     * Returns error response array on failure, null on success.
     */
    private function validatePassengers(array $passengers, int $expectedCount): ?array
    {
        if (count($passengers) !== $expectedCount) {
            return $this->errorResponse(
                'VALIDATION_ERROR',
                "Expected {$expectedCount} passenger(s), received " . count($passengers) . '.',
                'RETRY'
            );
        }

        $required = ['salutation', 'firstName', 'lastName', 'nationality', 'residenceCountry', 'email'];

        foreach ($passengers as $i => $p) {
            foreach ($required as $field) {
                if (empty($p[$field])) {
                    return $this->errorResponse(
                        'PASSENGER_VALIDATION_FAILED',
                        'Please provide passenger ' . $field . ' for passenger ' . ($i + 1) . '.',
                        'RETRY'
                    );
                }
            }
            if (! filter_var($p['email'], FILTER_VALIDATE_EMAIL)) {
                return $this->errorResponse(
                    'PASSENGER_VALIDATION_FAILED',
                    'Passenger ' . ($i + 1) . ' email format is invalid.',
                    'RETRY'
                );
            }
        }

        return null;
    }

    private function errorResponse(string $code, string $message, string $action): array
    {
        return [
            'success' => false,
            'error'   => $this->buildError($code, $message, $action, null),
            'meta'    => $this->buildMeta(null),
            'cached'  => false,
            'data'    => null,
        ];
    }

    private function buildError(string $code, string $message, string $action, ?string $details): array
    {
        return [
            'error_code'    => $code,
            'error_message' => $message,
            'error_details' => $details ?? '',
            'action'        => $action,
        ];
    }

    private function buildMeta(?int $companyId): array
    {
        return [
            'trace_id'   => app('dotw.trace_id'),
            'company_id' => $companyId,
            'timestamp'  => now()->toISOString(),
        ];
    }

    /**
     * Returns empty itinerary for error responses where we still need the data structure.
     */
    private function emptyItinerary(DotwPrebook $prebook, string $checkin, string $checkout): array
    {
        return [
            'hotel_code'          => $prebook->hotel_code,
            'hotel_name'          => $prebook->hotel_name ?? $prebook->hotel_code,
            'checkin'             => $checkin,
            'checkout'            => $checkout,
            'room_type'           => $prebook->room_type,
            'rate_basis'          => $prebook->room_rate_basis ?? '',
            'total_fare'          => (float) $prebook->total_fare,
            'currency'            => $prebook->original_currency ?: 'USD',
            'is_refundable'       => (bool) $prebook->is_refundable,
            'lead_guest_name'     => '',
            'customer_reference'  => '',
            'confirmation_number' => '',
        ];
    }
}
```

**Critical implementation rules:**

1. `action` field values MUST be valid `DotwErrorAction` enum members: `RETRY`, `RETRY_IN_30_SECONDS`, `RECONFIGURE_CREDENTIALS`, `RESEARCH`, `CANCEL`, `NONE`. Do NOT use `RESUBMIT` — it is NOT in the enum and will fail schema validation (Pitfall 2 from research).

2. `allocationDetails` passed to confirmBooking MUST be raw from `$prebook->allocation_details` — no trim, urlencode, or base64 (Pitfall 1 from research).

3. The supplementary audit log (BOOK-07) is wrapped in `try/catch(\Throwable)` — audit failure must NEVER break the booking response.

4. `PASSENGER_VALIDATION_FAILED` is used as error_code for passenger validation errors. This is a project-specific string, not a `DotwErrorCode` enum value. Use it as-is — the error_code field accepts string.

5. DotwService is instantiated inside `__invoke` with `$companyId` — same pattern as DotwBlockRates. Do NOT inject DotwService via constructor (per-request credential resolution).

6. For `RATE_UNAVAILABLE` detection: use `str_contains(strtolower($message), ...)` on the exception message — DOTW error codes in exceptions are not standardized, so string matching is the pragmatic approach (Low confidence area noted in research).

7. For the alternatives search, use `fail-silent` try/catch(\Throwable) — alternative search must never cascade into a new error if DOTW is having issues.

8. After successful booking creation, `$prebook->update(['expired_at' => now()])` inside DB::transaction prevents the same prebook from being used again.
