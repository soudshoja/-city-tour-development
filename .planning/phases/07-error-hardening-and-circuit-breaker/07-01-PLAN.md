<objective>
Create a typed timeout exception and wire it through DotwService::post() so resolvers can distinguish API timeouts from credential errors and generic API failures.

Purpose: ERROR-02 requires "Search taking too long, please try again" with action: retry — not the generic "Hotel search failed". The current DotwService catches all Exception uniformly in post(), so resolvers cannot distinguish a timeout from any other API failure. This plan adds DotwTimeoutException and reduces the timeout default from 120s to 25s.

Output: DotwTimeoutException class, patched DotwService::post(), updated config/dotw.php default. All existing resolvers (DotwSearchHotels, DotwGetRoomRates, DotwBlockRates, DotwGetCities) are updated to catch DotwTimeoutException before RuntimeException.
</objective>

   - PHPDoc: "Thrown by DotwService::post() when the DOTW API does not respond within the configured timeout window."

2. Patch `app/Services/DotwService.php` — in the `post()` private method:
   - Add `use Illuminate\Http\Client\ConnectionException;` import at the top of the file.
   - Add `use App\Exceptions\DotwTimeoutException;` import.
   - In the `post()` method catch chain, add a new catch BEFORE the existing `catch (Exception $e)`:
     ```php
     } catch (ConnectionException $e) {
         $this->logger->error('DOTW API timeout', [
             'timeout_seconds' => $this->timeout,
             'company_id' => $this->companyId,
         ]);
         throw new DotwTimeoutException(
             "DOTW API timeout after {$this->timeout}s",
             0,
             $e
         );
     } catch (Exception $e) {
         // existing catch block — unchanged
     ```
   - Do NOT modify any existing catch block logic.
   - Do NOT log credentials ($username, $passwordMd5, $companyCode) in the new catch block.

   - To: `'timeout' => env('DOTW_TIMEOUT', 25),`
   - Update the comment above to: "Maximum seconds to wait for a DOTW API response. DOTW SLA: 25 seconds (ERROR-02)."
   - `connect_timeout` stays at `env('DOTW_CONNECT_TIMEOUT', 30)` — unchanged.
