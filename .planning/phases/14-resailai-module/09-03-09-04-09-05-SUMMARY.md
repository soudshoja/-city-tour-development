## n8n Webhook Configuration

### n8n Webhook Setup
1. **Node Type:** Webhook
2. **Path:** `/resailai-process`
3. **Method:** POST
4. **Authentication:** None (Bearer token in payload)

### Payload Example
```json
{
    "document_id": "uuid-string",
    "supplier_id": 2,
    "company_id": 1,
    "agent_id": 5,
    "branch_id": 1,
    "file_path": "storage/app/company/supplier/files_unprocessed/filename.pdf",
    "callback_url": "https://laravel.example.com/api/modules/resailai/callback"
}
```

### Callback URL
- **Endpoint:** `POST /api/modules/resailai/callback`
- **Auth Method:** Bearer Token (RESAILAI_API_TOKEN)
- **Response:** JSON with document_id confirmation

## Deployment Steps

1. **Run Migration:**
   ```bash
   php artisan migrate
   ```

2. **Clear Caches:**
   ```bash
   php artisan config:clear
   php artisan route:clear
   php artisan cache:clear
   ```

3. **Set Environment Variables:**
   ```bash
   RESAILAI_API_TOKEN=your-secure-token-here
   N8N_WEBHOOK_URL=https://n8n.example.com/webhook/resailai-process
   ```

4. **Restart Queue Worker:**
   ```bash
   php artisan queue:restart
   php artisan queue:work --verbose
   ```

5. **Verify Routes:**
   ```bash
   php artisan route:list --path=resailai
   ```
