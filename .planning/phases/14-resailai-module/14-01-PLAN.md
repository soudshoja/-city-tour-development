## Objective

Create the foundational infrastructure for the ResailAI module: database migration for the feature flag, service provider for bootstrapping, config file for module settings, and routing configuration.

**Purpose:** Establish the core architecture that enables the module to be self-contained and loaded automatically by Laravel. This wave ensures the database structure exists and the module is properly integrated into Laravel's service container.

**Output:**
- Migration file with `auto_process_pdf` column
- Service provider registering module routes, config, and middleware
- Config file with sensible defaults
- routes/modules/resailai.php with callback route
