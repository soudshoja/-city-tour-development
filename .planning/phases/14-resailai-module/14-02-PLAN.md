## Objective

Create the Admin UI and Documentation for the ResailAI module: API key management, supplier feature flag toggle, and comprehensive developer documentation.

**Purpose:** Provide admin interface to manage ResailAI configuration and enable/disable PDF processing per supplier, plus documentation for n8n webhook setup and API callback configuration.

**Output:**
- Admin UI v1: API key generation and management
- Admin UI v2: Feature flag toggle per supplier
- Developer documentation: Webhook and callback configuration guide
