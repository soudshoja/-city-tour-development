    **1. Create VoucherService** at `app/Modules/DotwAI/Services/VoucherService.php`:
    Namespace: `App\Modules\DotwAI\Services`

    This service handles voucher formatting and delivery after booking confirmation. Vouchers are text-based WhatsApp messages (per Claude's discretion -- text vouchers are simpler and more reliable than PDF attachments for WhatsApp delivery).

    **Method `sendVoucher(DotwAIBooking $booking): bool`:**
    1. Format the voucher message: `$message = MessageBuilderService::formatVoucherMessage($booking)`.
    2. Determine recipient phone: `$phone = $booking->client_phone ?? $booking->agent_phone`.
    3. Send via WhatsApp:
       ```php
       $whatsapp = app(\App\Http\Controllers\WhatsappController::class);
       $result = $whatsapp->sendToResayil(
           $phone,
           $message,
           'Booking Confirmation | تأكيد الحجز',  // header
           'City Travelers',                        // footer
           null                                      // no buttons
       );
       ```
    4. Update booking: `$booking->update(['voucher_sent_at' => now()])`.
    5. Log: `Log::info('[DotwAI] Voucher sent', ['prebook_key' => $booking->prebook_key, 'phone' => $phone])`.
    6. Return true on success, false on failure (catch exceptions from sendToResayil).

    **Method `resendVoucher(DotwAIBooking $booking): bool`:**
    Same as sendVoucher but does NOT check voucher_sent_at (allows re-sending). Logs as "resent".

    **2. Add formatVoucherMessage to MessageBuilderService:**

    `public static function formatVoucherMessage(DotwAIBooking $booking): string`:

    Build a comprehensive bilingual voucher message. Include paymentGuaranteedBy per locked CONTEXT.md decision.

    Format:
    ```
    BOOKING CONFIRMATION | تأكيد الحجز
    ══════════════════════════════

    Booking Reference: {confirmation_no}
    الرقم المرجعي: {confirmation_no}
    ──────────────────────────────

    Hotel | الفندق: {hotel_name}
    Check-in | تسجيل الدخول: {check_in formatted as d M Y}
    Check-out | تسجيل الخروج: {check_out formatted as d M Y}
    ──────────────────────────────

    Guest(s) | الضيوف:
    {foreach guest in guest_details: "- {salutation} {first_name} {last_name}"}
    ──────────────────────────────

    Total | المجموع: {display_currency} {display_total_fare}
    Status | الحالة: Confirmed | مؤكد
    {if payment_guaranteed_by:}
    Payment Guaranteed By: {payment_guaranteed_by}
    {endif}
    ──────────────────────────────

    {if is_refundable:}
    Free cancellation until {cancellation_deadline formatted}
    الإلغاء المجاني حتى {cancellation_deadline formatted}
    {else:}
    Non-Refundable (APR) | غير قابل للاسترداد
    {endif}

    ══════════════════════════════
    City Travelers | سيتي ترافلرز
    ```

    Handle null guest_details gracefully (show "Guest" if no details).
    Handle null cancellation_deadline (show "See cancellation policy" instead).
  </action>
