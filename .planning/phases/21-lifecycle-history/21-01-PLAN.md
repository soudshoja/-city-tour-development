**Step 2: Create LifecycleService.php**

Location: `app/Modules/DotwAI/Services/LifecycleService.php`

Purpose: Deadline detection logic for scheduler (queries, not API calls).

Methods:
- `findBookingsDueForReminder(): Collection` — Query bookings with upcoming deadlines (3/2/1 day window), refundable only (is_apr = false), status = 'confirmed', reminder_sent_at IS NULL. Return collection.
- `findBookingsWithPassedDeadline(): Collection` — Query bookings with cancellation_deadline < now(), status = 'confirmed', auto_invoiced_at IS NULL. Return collection.
- `markReminderSent(int $bookingId): void` — Update booking reminder_sent_at = now(). Used after reminder job succeeds.

Implementation:
```php
class LifecycleService {
    public function findBookingsDueForReminder(): Collection {
        return DotwAIBooking::where('status', 'confirmed')
            ->where('is_apr', false)
            ->whereNull('reminder_sent_at')
            ->where('cancellation_deadline', '>=', now())
            ->where('cancellation_deadline', '<=', now()->addDays(3))
            ->get();
    }

    public function findBookingsWithPassedDeadline(): Collection {
        return DotwAIBooking::where('status', 'confirmed')
            ->where('cancellation_deadline', '<', now())
            ->whereNull('auto_invoiced_at')
            ->get();
    }

    public function markReminderSent(int $bookingId): void {
        DotwAIBooking::findOrFail($bookingId)->update(['reminder_sent_at' => now()]);
    }
}
```

**Step 1: Create SendReminderJob.php**

Location: `app/Modules/DotwAI/Jobs/SendReminderJob.php`

This job sends a WhatsApp reminder and marks the booking's reminder_sent_at field. It dispatches from the scheduler with a specific booking ID.

```php
<?php

declare(strict_types=1);

namespace App\Modules\DotwAI\Jobs;

use App\Modules\DotwAI\Models\DotwAIBooking;
use App\Modules\DotwAI\Services\LifecycleService;
use App\Modules\DotwAI\Services\MessageBuilderService;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Foundation\Queue\Queueable;
use Illuminate\Support\Facades\Log;
use Throwable;

class SendReminderJob implements ShouldQueue
{
    use Queueable;

    public int $tries = 3;
    public array $backoff = [30, 120];  // 30s, 2m
    public int $timeout = 10;

    public function __construct(private int $bookingId) {}

    public function handle(LifecycleService $service): void
    {
        $booking = DotwAIBooking::find($this->bookingId);

        if (!$booking || !empty($booking->reminder_sent_at)) {
            return;  // Already processed or deleted
        }

        // Calculate days until deadline
        $daysLeft = now()->diffInDays($booking->cancellation_deadline, false);

        // Send WhatsApp reminder via Resayil (existing WhatsappController)
        $message = MessageBuilderService::formatReminderMessage($booking, abs($daysLeft));
        $phone = $booking->client_phone ?? $booking->agent_phone;

        try {
            // Call WhatsappController's sendToResayil method
            app(\App\Http\Controllers\WhatsappController::class)
                ->sendToResayil($phone, $message);

            // Mark as sent only after successful dispatch
            $service->markReminderSent($this->bookingId);

            Log::info('[DotwAI] Reminder sent', [
                'booking_id' => $this->bookingId,
                'phone' => $phone,
                'days_left' => $daysLeft,
            ]);
        } catch (Throwable $e) {
            Log::warning('[DotwAI] Reminder send failed, will retry', [
                'booking_id' => $this->bookingId,
                'error' => $e->getMessage(),
            ]);
            throw $e;  // Trigger retry via $backoff
        }
    }

    public function failed(Throwable $exception): void
    {
        Log::error('[DotwAI] Reminder job failed after retries', [
            'booking_id' => $this->bookingId,
            'error' => $exception->getMessage(),
        ]);
        // Booking remains without reminder_sent_at; scheduler will retry next cycle
    }
}
```

**Step 2: Create AutoInvoiceDeadlineJob.php**

Location: `app/Modules/DotwAI/Jobs/AutoInvoiceDeadlineJob.php`

This job handles the deadline-pass flow: creates Invoice + JournalEntry via AccountingService, sends voucher, marks auto_invoiced_at.

Key pattern: Use explicit company_id for all Account/JournalEntry queries (from Phase 20).

```php
<?php

declare(strict_types=1);

namespace App\Modules\DotwAI\Jobs;

use App\Modules\DotwAI\Models\DotwAIBooking;
use App\Modules\DotwAI\Services\AccountingService;
use App\Modules\DotwAI\Services\MessageBuilderService;
use App\Modules\DotwAI\Services\VoucherService;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Foundation\Queue\Queueable;
use Illuminate\Support\Facades\DB;
use Illuminate\Support\Facades\Log;
use Throwable;

class AutoInvoiceDeadlineJob implements ShouldQueue
{
    use Queueable;

    public int $tries = 3;
    public array $backoff = [60, 300];  // 1m, 5m
    public int $timeout = 30;

    public function __construct(private int $bookingId) {}

    public function handle(): void
    {
        $booking = DotwAIBooking::find($this->bookingId);

        if (!$booking || !empty($booking->auto_invoiced_at)) {
            return;  // Already invoiced or deleted
        }

        // Re-verify deadline really passed (prevent clock skew issues)
        if ($booking->cancellation_deadline >= now()) {
            Log::warning('[DotwAI] Deadline not yet passed, deferring auto-invoice', [
                'booking_id' => $this->bookingId,
                'deadline' => $booking->cancellation_deadline,
            ]);
            // Reschedule this job for later
            throw new \Exception('Deadline not yet passed');
        }

        try {
            DB::transaction(function () use ($booking) {
                // Create accounting entries (Invoice + JournalEntry)
                $accountingService = new AccountingService();
                $accountingService->createAutoInvoiceForDeadline($booking);

                // Send voucher via WhatsApp
                $voucherService = new VoucherService();
                $voucherService->sendVoucher($booking);

                // Mark as invoiced
                $booking->update(['auto_invoiced_at' => now()]);

                Log::info('[DotwAI] Auto-invoiced after deadline', [
                    'booking_id' => $booking->id,
                    'hotel' => $booking->hotel_name,
                    'deadline' => $booking->cancellation_deadline,
                ]);
            });
        } catch (Throwable $e) {
            Log::error('[DotwAI] Auto-invoice failed', [
                'booking_id' => $this->bookingId,
                'error' => $e->getMessage(),
            ]);
            throw $e;  // Trigger retry
        }
    }

    public function failed(Throwable $exception): void
    {
        Log::critical('[DotwAI] Auto-invoice job dead-lettered', [
            'booking_id' => $this->bookingId,
            'error' => $exception->getMessage(),
        ]);
    }
}
```

Notes on implementation:
- Both jobs use dependency injection (LifecycleService, others auto-wired by Laravel)
- SendReminderJob catches exceptions from WhatsappController and re-throws to trigger backoff retry
- AutoInvoiceDeadlineJob re-verifies deadline hasn't changed due to clock skew
- Both jobs have explicit failed() hooks for dead-letter logging
- Backoff arrays use exponential delays (no magic 5min constants)

**Step 1: Create ProcessDeadlinesCommand.php**

Location: `app/Modules/DotwAI/Commands/ProcessDeadlinesCommand.php`

This is the scheduler entry point. It runs daily, queries for bookings due for reminders and with passed deadlines, dispatches queue jobs, and logs results.

```php
<?php

declare(strict_types=1);

namespace App\Modules\DotwAI\Commands;

use App\Modules\DotwAI\Jobs\AutoInvoiceDeadlineJob;
use App\Modules\DotwAI\Jobs\SendReminderJob;
use App\Modules\DotwAI\Services\LifecycleService;
use Illuminate\Console\Command;
use Illuminate\Support\Facades\Log;

class ProcessDeadlinesCommand extends Command
{
    protected $signature = 'dotwai:process-deadlines
                        {--batch-size=50 : Number of bookings to process per job dispatch}';

    protected $description = 'Process booking deadlines: send reminders at 3/2/1 days, auto-invoice when deadline passes';

    public function handle(LifecycleService $service): int
    {
        $batchSize = (int) $this->option('batch-size');

        $this->info('Starting deadline processor...');

        // Find and dispatch reminders for bookings due
        $reminderCount = 0;
        $bookingsDue = $service->findBookingsDueForReminder();

        foreach ($bookingsDue as $booking) {
            SendReminderJob::dispatch($booking->id);
            $reminderCount++;
        }

        if ($reminderCount > 0) {
            $this->info("Dispatched {$reminderCount} reminder job(s)");
            Log::info('[DotwAI] Scheduler dispatched reminders', ['count' => $reminderCount]);
        }

        // Find and dispatch auto-invoicing for passed deadlines
        $invoiceCount = 0;
        $passedDeadlines = $service->findBookingsWithPassedDeadline();

        foreach ($passedDeadlines as $booking) {
            AutoInvoiceDeadlineJob::dispatch($booking->id);
            $invoiceCount++;
        }

        if ($invoiceCount > 0) {
            $this->info("Dispatched {$invoiceCount} auto-invoice job(s)");
            Log::info('[DotwAI] Scheduler dispatched auto-invoices', ['count' => $invoiceCount]);
        }

        $this->info('Deadline processor completed');
        Log::info('[DotwAI] Deadline processor cycle complete', [
            'reminders_dispatched' => $reminderCount,
            'invoices_dispatched' => $invoiceCount,
        ]);

        return self::SUCCESS;
    }
}
```
