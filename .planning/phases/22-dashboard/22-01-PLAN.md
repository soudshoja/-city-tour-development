Public properties (stats cards):

- `public int $bookingsToday = 0;`
- `public int $errorsToday = 0;`

Public properties (chart data, serialized as JSON for Blade):
- `public array $bookingTrendDates = [];`  — last 14 calendar dates (Y-m-d strings, oldest first)
- `public array $bookingTrendCounts = [];` — booking count per date
- `public array $operationLabels = ['search', 'rates', 'block', 'book'];`
- `public array $operationCounts = [0, 0, 0, 0];` — last 7 days count per operation

Public properties (recent API log table):
- `public array $recentLogs = [];` — last 25 dotw_audit_log rows as plain arrays (id, company_id, resayil_message_id, operation_type, has_empty_response, created_at formatted)

Method `isSuperAdmin(): bool` — returns `Auth::user()->role_id === Role::ADMIN`

Method `mount(): void` — calls `$this->refreshMetrics()`

Method `refreshMetrics(): void`:
1. Build base booking scope: `DotwAIBooking::query()->when(!$this->isSuperAdmin(), fn($q) => $q->where('company_id', Auth::user()->company_id))`
2. Compute `$this->totalBookings` = count()
3. Compute `$this->bookingsToday` = whereDate('created_at', today())->count()
4. Compute `$this->errorsToday` = whereDate('created_at', today())->whereIn('status', ['failed', 'expired'])->count()
5. Compute `$this->activePrebooks` = whereIn('status', ['prebooked', 'pending_payment'])->count()

6. Build booking trend (last 14 days):
   ```php
   $dates = collect(range(13, 0))->map(fn($d) => now()->subDays($d)->format('Y-m-d'));
   $trendData = DotwAIBooking::query()
       ->when(!$this->isSuperAdmin(), fn($q) => $q->where('company_id', Auth::user()->company_id))
       ->whereDate('created_at', '>=', now()->subDays(13))
       ->selectRaw('DATE(created_at) as date, COUNT(*) as count')
       ->groupBy('date')
       ->pluck('count', 'date');
   $this->bookingTrendDates = $dates->toArray();
   $this->bookingTrendCounts = $dates->map(fn($d) => (int) ($trendData[$d] ?? 0))->toArray();
   ```

7. Build operation breakdown (last 7 days):
   ```php
   $opData = DotwAuditLog::query()
       ->when(!$this->isSuperAdmin(), fn($q) => $q->where('company_id', Auth::user()->company_id))
       ->whereDate('created_at', '>=', now()->subDays(6))
       ->selectRaw('operation_type, COUNT(*) as count')
       ->groupBy('operation_type')
       ->pluck('count', 'operation_type');
   $this->operationCounts = collect($this->operationLabels)->map(fn($op) => (int) ($opData[$op] ?? 0))->toArray();
   ```

8. Build recent API log (last 25 rows, most recent first):
   ```php
   $this->recentLogs = DotwAuditLog::query()
       ->when(!$this->isSuperAdmin(), fn($q) => $q->where('company_id', Auth::user()->company_id))
       ->orderByDesc('created_at')
       ->limit(25)
       ->get()
       ->map(fn($log) => [
           'id'               => $log->id,
           'company_id'       => $log->company_id,
           'message_id'       => $log->resayil_message_id ?? '—',
           'operation_type'   => $log->operation_type,
           'has_empty_response' => empty($log->response_payload),
           'created_at'       => $log->created_at->format('Y-m-d H:i:s'),
       ])
       ->toArray();
   ```

9. After all data is computed, dispatch: `$this->dispatch('dashboardMetricsUpdated');`

Method `render(): \Illuminate\View\View` — returns `view('livewire.admin.dotw-dashboard-tab')`

NOTE: Do not pass variables to the view — the Blade template accesses component public properties directly via `$this->` (Livewire 3 standard; public properties are auto-available in view).

```blade
@script
<script>
    let trendChart = null;
    let opsChart = null;

    function initCharts() {
        const isDark = document.documentElement.classList.contains('dark');
        const textColor = isDark ? '#9ca3af' : '#6b7280';

        // Destroy existing charts before re-init (prevents duplication on poll)
        if (trendChart) { trendChart.destroy(); trendChart = null; }
        if (opsChart) { opsChart.destroy(); opsChart = null; }

        const trendEl = document.getElementById('dotw-bookings-trend-chart');
        const opsEl = document.getElementById('dotw-operations-chart');
        if (!trendEl || !opsEl) return;

        trendChart = new ApexCharts(trendEl, {
            chart: { type: 'line', height: 220, toolbar: { show: false }, background: 'transparent' },
            series: [{ name: 'Bookings', data: @json($bookingTrendCounts) }],
            xaxis: { categories: @json($bookingTrendDates), labels: { style: { colors: textColor, fontSize: '10px' } } },
            yaxis: { labels: { style: { colors: textColor } } },
            colors: ['#3b82f6'],
            stroke: { curve: 'smooth', width: 2 },
            grid: { borderColor: isDark ? '#374151' : '#e5e7eb' },
            theme: { mode: isDark ? 'dark' : 'light' },
        });
        trendChart.render();

        opsChart = new ApexCharts(opsEl, {
            chart: { type: 'bar', height: 220, toolbar: { show: false }, background: 'transparent' },
            series: [{ name: 'Calls', data: @json($operationCounts) }],
            xaxis: { categories: @json($operationLabels), labels: { style: { colors: textColor } } },
            yaxis: { labels: { style: { colors: textColor } } },
            colors: ['#f59e0b'],
            plotOptions: { bar: { borderRadius: 4 } },
            grid: { borderColor: isDark ? '#374151' : '#e5e7eb' },
            theme: { mode: isDark ? 'dark' : 'light' },
        });
        opsChart.render();
    }

    // Initial render
    initCharts();

    // Re-render after every Livewire update (poll refresh)
    $wire.on('dashboardMetricsUpdated', () => {
        initCharts();
    });
</script>
@endscript
```
