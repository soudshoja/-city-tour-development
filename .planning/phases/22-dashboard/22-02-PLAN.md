Public properties:

- `public string $filterDateFrom = '';`
- `public string $filterDateTo = '';`

`$queryString` array (same pattern as DotwAuditLogIndex):
```php
protected $queryString = [
    'filterStatus'   => ['except' => ''],
    'filterDateFrom' => ['except' => ''],
    'filterDateTo'   => ['except' => ''],
];
```

Methods:

`isSuperAdmin(): bool` — returns `Auth::user()->role_id === Role::ADMIN`

`toggleRow(int $id): void` — sets `$this->expandedRow = $this->expandedRow === $id ? null : $id`

`resetFilters(): void` — resets all filter properties to '', expandedRow to null, calls `$this->resetPage()`

`lifecycleStages(DotwAIBooking $booking): array` — returns an array of stage arrays representing the horizontal stepper. Each stage:
```php
['label' => string, 'reached' => bool, 'failed' => bool, 'timestamp' => ?string]
```

Stages and their logic:
- Stage 0 — "Prebooked": always reached=true, timestamp = $booking->created_at->format('Y-m-d H:i')
- Stage 1 — "Payment": reached = ($booking->payment_status === 'paid' || in_array($booking->track, ['b2b'])), failed = ($booking->status === 'failed' && $booking->payment_status !== 'paid'), timestamp = null (no dedicated timestamp column; show "Paid" or "N/A")
- Stage 2 — "Confirmed": reached = in_array($booking->status, ['confirmed', 'cancellation_pending', 'cancelled']), failed = ($booking->status === 'failed'), timestamp = null (no confirmed_at column; show confirmation_no if present)
- Stage 3 — "Voucher Sent": reached = ($booking->voucher_sent_at !== null), failed = false, timestamp = $booking->voucher_sent_at?->format('Y-m-d H:i')
- Stage 4 — "Cancelled": reached = ($booking->status === 'cancelled'), failed = true (always red when reached), timestamp = null

Note: For B2B track (no payment step), Stage 1 is auto-marked as reached=true with label "Credit" to indicate credit deduction instead of payment.

Implement this as a public method returning the array; the Blade will call `$this->lifecycleStages($booking)` per row.

`render(): \Illuminate\View\View`:
```php
$bookings = DotwAIBooking::query()
    ->when(!$this->isSuperAdmin(), fn($q) => $q->where('company_id', Auth::user()->company_id))
    ->when($this->filterStatus, fn($q) => $q->where('status', $this->filterStatus))
    ->when($this->filterDateFrom, fn($q) => $q->whereDate('created_at', '>=', $this->filterDateFrom))
    ->when($this->filterDateTo, fn($q) => $q->whereDate('created_at', '<=', $this->filterDateTo))
    ->orderByDesc('created_at')
    ->paginate(25);

return view('livewire.admin.dotw-booking-lifecycle-tab', [
    'bookings'    => $bookings,
    'isSuperAdmin' => $this->isSuperAdmin(),
]);
```
