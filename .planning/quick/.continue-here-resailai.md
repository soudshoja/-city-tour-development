<context>
The goal is a production-safe, self-contained module that handles automated document extraction for ALL suppliers and ALL task types.

**Architecture:**
```
File upload → storage/app/{company}/{supplier}/files_unprocessed/
→ ResailAIPdfProcessor middleware (checks auto_process_pdf flag)
→ ProcessDocumentJob (queue)
→ Downloads file + sends to n8n webhook with Bearer token
→ Receives extracted data from AI
→ ResailAITaskBridge calls TaskWebhook::webhook()
→ Task created via existing TaskWebhook pipeline
```

**API Key Flow:**
```
Admin visits /admin/resailai/keys
→ Clicks "Generate API Key"
→ System creates encrypted record + copies to clipboard
→ Admin pastes key into n8n webhook configuration
→ n8n sends: POST /resailai/callback with Bearer token
```

Complete research: `.planning/quick/resailai-module-research.md`
